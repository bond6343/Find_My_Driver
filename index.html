One-time setup: this app needs a free Firebase database to pass locations back and forth. Follow these steps once, ever:

  1. Go to console.firebase.google.com and sign in with any Google account
  2. Click Add project, name it anything, finish creation
  3. In the left menu, click Build → Realtime Database → Create Database → choose any location → start in test mode
  4. Click the gear icon next to "Project Overview" → Project settings
  5. Scroll to "Your apps" → click the </> (web) icon → register an app (any nickname) → it'll show a code block starting with const firebaseConfig = {'{'}...{'}'}
  6. Copy that whole config block and paste it below, then tap Test
